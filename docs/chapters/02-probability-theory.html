12  A Short Recap of Probability Theory

This section provides a concise review of the basic concepts of probability theory that will be used throughout these lecture notes on Bayesian inference.

12.1 Probability Spaces and Axioms

A probability space is a triple \((\Omega, \mathcal{F}, \mathbb{P})\), where:

  • \(\Omega\) is the sample space, i.e. the set of all possible outcomes;
  • \(\mathcal{F}\) is a \(\sigma\)-algebra of subsets of \(\Omega\), (the set of all possible outcomes of an experiment) whose elements are called events 1
  • \(\mathbb{P}: \mathcal{F} \rightarrow[0,1]\) is a probability measure.

The probability measure satisfies the Kolmogorov axioms:

  1. Non-negativity: \(\mathbb{P}(A) \geq 0\) for all \(A \in \mathcal{F}\);
  2. Normalization: \(\mathbb{P}(\Omega)=1\);
  3. Countable additivity: for any countable collection of disjoint events \(\left\{A_{i}\right\}_{i=1}^{\infty}\),

\[ \mathbb{P}\left(\bigcup_{i=1}^{\infty} A_{i}\right)=\sum_{i=1}^{\infty} \mathbb{P}\left(A_{i}\right) \]

12.2 Events and Set Operations

Events (elements of the \(\sigma\)-algebra \(\mathcal{F}\)) can be combined using standard set-theoretic operations, which correspond to logical operations. The properties of the associated probabilities follow from the Kolmogorov axioms.

Union (Logical OR). Given two events \(A, B\), the union

\[ A \cup B=\{\omega \in \Omega: \omega \in A \text { or } \omega \in B\} \]

represents the event that at least one of the events \(A\) or \(B\) occurs.

The probability of the union is given by the inclusion-exclusion principle:

\[ \mathbb{P}(A \cup B)=\mathbb{P}(A)+\mathbb{P}(B)-\mathbb{P}(A \cap B) \]

If \(A\) and \(B\) are disjoint (mutually exclusive), i.e. \(A \cap B=\varnothing\) is the empty set, then

\[ \mathbb{P}(A \cup B)=\mathbb{P}(A)+\mathbb{P}(B) \]

Intersection (Logical AND). The intersection

\[ A \cap B=\{\omega \in \Omega: \omega \in A \text { and } \omega \in B\} \]

corresponds to the event that both \(A\) and \(B\) occur. Its probability is directly related to conditional probability:

\[ \mathbb{P}(A \cap B)=\mathbb{P}(A \mid B) \mathbb{P}(B)=\mathbb{P}(B \mid A) \mathbb{P}(A) \tag{6} \]

whenever the conditioning probabilities are well-defined. The conditional probability \(\mathbb{P}(B \mid A)\) is the probability of event \(B\) given that \(A\) is taken as true/occurred.

Complement (Negation). The complement (or negation) of an event \(A\) is

\[ \bar{A}=\Omega \backslash A, \]

representing the event that \(A\) does not occur. By normalization,

\[ \mathbb{P}(\bar{A})=1-\mathbb{P}(A) . \]

Set Difference. The difference between two events is

\[ A \backslash B=A \cap \bar{B}, \]

corresponding to the event that \(A\) occurs but \(B\) does not. Its probability is

\[ \mathbb{P}(A \backslash B)=\mathbb{P}(A)-\mathbb{P}(A \cap B) . \]

De Morgan’s Laws. For any two events \(A\) and \(B\),

\[ \overline{(A \cup B)}=\bar{A} \cap \bar{B}, \quad \overline{(A \cap B)}=\bar{A} \cup \bar{B} . \]

These identities are often useful when computing probabilities involving negations.

Monotonicity. If \(A \subseteq B\), then

\[ \mathbb{P}(A) \leq \mathbb{P}(B) . \]

In particular, probabilities are bounded as

\[ 0 \leq \mathbb{P}(A) \leq 1 . \]

Partition of the Sample Space. A collection of events \(\left\{A_{i}\right\}_{i=1}^{n}\) is called a partition of \(\Omega\) if:

  1. \(A_{i} \cap A_{j}=\varnothing\) for \(i \neq j\),
  2. \(\bigcup_{i=1}^{n} A_{i}=\Omega\).

In this case,

\[ \sum_{i=1}^{n} \mathbb{P}\left(A_{i}\right)=1 . \]

Partitions play a central role in Bayesian inference, where they often represent mutually exclusive hypotheses or models. For \(n=2\), we go back to a single event and its complement.

12.3 Random Variables

A real-valued random variable \(X\) is a measurable function

\[ X: \Omega \rightarrow \mathbb{R} . \]

Rather than working directly with outcomes \(\omega \in \Omega\), probability theory is typically formulated in terms of random variables and their distributions. A random variable thus assigns a real number to each outcome, i.e. to each result of an experiment. Basically, \(X\) tells you which quantity you measure in your experiment.

Note the important distinction: in the following the random variable \(X\) is a function, mapping an event to real numbers, while \(x\) is the particular numerical value that it can take (e.g., in one realization).

12.4 Probability Mass and Density Functions

If \(X\) is a discrete random variable taking values in a countable set \(\left\{x_{i}\right\}\), its probability mass function (pmf) is

\[ p_{X}\left(x_{i}\right)=\mathbb{P}\left(X=x_{i}\right), \quad \sum_{i} p_{X}\left(x_{i}\right)=1 \]

If \(X\) is a continuous random variable, it is characterized by a probability density function (pdf) \(f_{X}(x)\) such that

\[ \mathbb{P}(a \leq X \leq b)=\int_{a}^{b} f_{X}(x) d x, \quad \int_{-\infty}^{\infty} f_{X}(x) d x=1 \]

Probabilities at a single point vanish in the continuous case:

\[ \mathbb{P}(X=x)=0 \]

it only makes sense to ask the probability that \(X\) follows within an (infinitesimal) interval.

12.5 Cumulative Distribution Function

The cumulative distribution function (CDF) of a random variable \(X\) is defined as

\[ F_{X}(x)=\mathbb{P}(X \leq x) \]

It is non-decreasing, right-continuous, and satisfies

\[ \lim _{x \rightarrow-\infty} F_{X}(x)=0, \quad \lim _{x \rightarrow+\infty} F_{X}(x)=1 \]

If \(X\) admits a density \(f_{X}\), then

\[ F_{X}(x)=\int_{-\infty}^{x} f_{X}(t) d t, \quad f_{X}(x)=\frac{d}{d x} F_{X}(x) \]

Instead, for the discrete random variable X with \(\operatorname{pmf} p_{X}\) the CDF reads

\[ F_{X}(x)=\sum_{y \leq x} P(X=y)=\sum_{i: x_{i} \leq x} p_{X}\left(x_{i}\right) . \tag{7} \]

In this case the CDF is made up of jumps of value \(p_{X}\left(x_{i}\right)=F\left(x_{i}\right)-F\left(x_{i-1}\right)\).

12.6 Joint Distributions

Given two random variables \(X\) and \(Y\), their joint distribution associated to \(P(X \cap Y)\) is described by:

  • a joint \(\operatorname{pmf} p_{X, Y}(x, y)\) in the discrete case;
  • a joint pdf \(f_{X, Y}(x, y)\) in the continuous case.

Marginal distributions are obtained by summation or integration:

\[ p_{X}(x)=\sum_{y} p_{X, Y}(x, y), \quad f_{X}(x)=\int f_{X, Y}(x, y) d y \]

12.7 Conditional Probability and Conditional Distributions

For random variables, the conditional distribution of \(X\) given \(Y=y\) is defined by

\[ p_{X \mid Y}(x \mid y)=\frac{p_{X, Y}(x, y)}{p_{Y}(y)}, \quad f_{X \mid Y}(x \mid y)=\frac{f_{X, Y}(x, y)}{f_{Y}(y)} \tag{8} \]

whenever the denominator is non-zero. The joint distribution factorizes as

\[ p_{X, Y}(x, y)=p_{X \mid Y}(x \mid y) p_{Y}(y), \quad f_{X, Y}(x, y)=f_{X \mid Y}(x \mid y) f_{Y}(y) \]

12.8 Expectation and Moments

The expectation (mean) of a random variable \(X\) is defined as

\[ \mathbb{E}[X]= \begin{cases}\sum_{x} x p_{X}(x), & \text { discrete case } \\ \int x f_{X}(x) d x, & \text { continuous case. }\end{cases} \]

More generally, for any function \(g(X)\), the expectation reads

\[ \mathbb{E}[g(X)]=\sum_{x} g(x) p_{X}(x) \quad \text { or } \quad \mathbb{E}[g(X)]=\int g(x) f_{X}(x) d x \]

The \(n\)-th raw moment (i.e., moment about zero) of \(X\) is

\[ \mu_{n}^{\prime} \equiv \mathbb{E}\left[X^{n}\right] \]

The \(n\)-th central moment (i.e., moment about the mean) of \(X\) is

\[ \mu_{n} \equiv \mathbb{E}\left[(X-\mathbb{E}[X])^{n}\right] \]

and provide clearer information about the distribution’s shape. Note that \(\mu_{1}=0\), while the variance is the \(n=2\) central moment, given by

\[ \mu_{2} \equiv \operatorname{Var}(X)=\mathbb{E}\left[(X-\mathbb{E}[X])^{2}\right]=\mathbb{E}\left[X^{2}\right]-(\mathbb{E}[X])^{2} \]

The standard deviation is the \(\sqrt{\mu_{2}} \equiv \sigma\). Finally, it is customary to define the standardized moment of degree \(n\) as \(\mu_{n} / \sigma^{n}\) that is, the ratio of the \(n\)-th central moment to the \(n\)-th power of the standard deviation. It offers a scale invariant, dimensionless measure for characteristics of a distribution.

12.9 Moment and Cumulant Generating Functions

The moment generating function (MGF) of \(X\) is defined as

\[ M_{X}(t)=\mathbb{E}\left[e^{t X}\right] \]

when it exists in a neighborhood of \(t=0\). Then derivatives of \(M_{X}(t)\) yields the \(n\)th central moment of \(X\):

\[ \left.\frac{d^{n} M_{X}(t)}{d t^{n}}\right|_{t=0}=\mu_{n}^{\prime} \tag{9} \]

The cumulant generating function (CGF) is

\[ K_{X}(t)=\log M_{X}(t) \]

The \(n\)-th cumulant \(\kappa_{n}\) is obtained by

\[ \kappa_{n}=\left.\frac{d^{n} K_{X}(t)}{d t^{n}}\right|_{t=0} \]

The first two cumulants coincide with the mean and the variance:

\[ \kappa_{1}=\mu_{1}^{\prime}, \quad \kappa_{2}=\mu_{2} \]

Cumulants play a central role in Bayesian inference and statistical mechanics, as characterization of fluctuations and correlations.

Note that MGF and CGF may not exist because the integral (sum) defining the expectation value need not converge absolutely. In this case, one can make the replacement \(t \mapsto i t\) defining the characteristic function

\[ \varphi_{X}(t)=\mathbb{E}\left[e^{i t X}\right] \]

whose derivatives give the central moments, \(\left.\frac{d^{n} \varphi_{X}(t)}{d t^{n}}\right|_{t=0}=i^{n} \mu_{n}^{\prime}\).

12.10 An important comment

All probabilities are always conditional ones! If \(I\) is the background condition, related to information we have, the probability of an event \(A\) is of the form \(P(A \mid I)\). It makes no sense to talk about the probability of the truth of the statement \(A\) without being explicit about the conditions upon which the assignment of probability is based.

12.11 Bayes’ formula

We start from that from the definition of joint probability (6). The key observation is that the joint probability \(\mathbb{P}(A \cap B)\) is symmetric in \(A\) and \(B\). Therefore, provided that \(\mathbb{P}(A)>0\) and \(\mathbb{P}(B)>0\), we may also write

\[ \mathbb{P}(A \cap B)=\mathbb{P}(B \mid A) \mathbb{P}(A) \tag{10} \]

Equating (6) and (10) yields

\[ \mathbb{P}(A \mid B) \mathbb{P}(B)=\mathbb{P}(B \mid A) \mathbb{P}(A) \]

that combine together to give Bayes’ formula:

\[ \mathbb{P}(A \mid B)=\frac{\mathbb{P}(B \mid A) \mathbb{P}(A)}{\mathbb{P}(B)} \tag{11} \]

A frequently used generalization arises when \(\left\{A_{i}\right\}_{i=1}^{n}\) is a partition of \(\Omega\), i.e. \(A_{i} \cap A_{j}=\varnothing\) for \(i \neq j\) and \(\bigcup_{i} A_{i}=\Omega\), with \(\mathbb{P}\left(A_{i}\right)>0\). Using countable additivity,

\[ \mathbb{P}(B)=\sum_{i=1}^{n} \mathbb{P}\left(B \cap A_{i}\right)=\sum_{i=1}^{n} \mathbb{P}\left(B \mid A_{i}\right) \mathbb{P}\left(A_{i}\right) \]

and substituting this into 11 gives

\[ \mathbb{P}\left(A_{k} \mid B\right)=\frac{\mathbb{P}\left(B \mid A_{k}\right) \mathbb{P}\left(A_{k}\right)}{\sum_{i=1}^{n} \mathbb{P}\left(B \mid A_{i}\right) \mathbb{P}\left(A_{i}\right)} \tag{12} \]

In data analysis and inference, \(A\) are parameters \(\theta\) that define the model (or the models \(H\) that we want to test) of the phenomenon, \(B\) are the data \(D\) (or \(X\)) that we get from the experiment. This form emphasizes the interpretation of Bayes’ formula as an update rule, where prior probabilities are revised in light of observed information, \(B\). Note that the normalization \(\mathbb{P}(B)\) is a constant independent of \(A_{k}\) but has been written in terms of the data \(B\) conditioned on the parameters/model.

If, given a likelihood function \(p(B \mid A)\), the posterior distribution \(p(A \mid B)\) is in the same probability distribution family as the prior probability distribution \(p(A)\), the prior and posterior are then called conjugate distributions with respect to that likelihood function and the prior is called a conjugate prior for the likelihood function \(p(B \mid A)\). A conjugate prior is an algebraic convenience, giving a closed-form expression for the posterior—numerical integration may be otherwise necessary.

Data inference is a slide down from the summit of Bayes’ formula.

  1. A \(\sigma\)-algebra is the mathematical structure that tells you which subsets of a sample space \(\Omega\) are allowed to be called events, i.e. which questions about the outcome are meaningful and can be assigned probabilities. An event is a yes/no question about the outcome (“Did the outcome fall in this set?” or “Did this condition occur?”). A \(\sigma\)-algebra is the collection of all such questions that you decide to be able to ask, and that remain meaningful when you combine them using logic (AND, OR, NOT).
